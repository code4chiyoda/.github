- 実用主義: 大まかな合意とすぐに動く実装が最重要です。いつか世界を変える政策よりも、いま死にそうな自分をただちに救う方法に興味があります。
- 文書化: その余裕があれば、文書を残します。いつか困難に直面する子どもたちが、迷ってしまわないように。
